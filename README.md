# From the Storm to the Fire

Working manuscript repository for **From the Storm to the Fire** by Daniel Lingar — a memoir of survival, truth, and the spaces in between.

## About This Project

This repository serves as the central hub for the book's ongoing development, tracking structural revisions, developmental edits, and archival materials as the project moves toward completion.

## Repository Structure

| Directory | Purpose |
|-----------|---------|
| `/manuscript` | Completed core chapters and sequential narrative tracks organized by part |
| `/drafts` | Evolutionary sketches, developmental edits, and raw chapter layouts |
| `/appendices` | Supplementary document registries, historical profiles, and source verifications |
| `/notes` | Structural roadmaps, thematic reference frameworks, and revision timelines |

## Manuscript Contents

The manuscript is organized into six parts, plus introduction, preface, prologue, and appendices:

**Part I: The Foundations** — Chapters 1-3
**Part II: Entanglement and Escalation** — Chapters 4-5
**Part III: Adrift** — Chapters 6-7
**Part IV: Decoding the Fire** — Chapters 8-13
**Part V: Reckoning and Rising** — Chapters 14-17
**Part VI: Forging the Flame** — Chapters 18-22

See `toc.md` for the complete table of contents.

## Current Status

The manuscript is actively under development. Some chapters are in final form, while others are in developmental stages. Placeholder files indicate sections pending content addition.

## How to Navigate

- **For the full manuscript structure:** See `toc.md`

- **For supporting materials:** Check `/appendices` and `/notes` directories
